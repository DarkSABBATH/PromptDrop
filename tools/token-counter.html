What are tokens?

Not quite words

Tokens are chunks of text — roughly 3–4 characters each. "hamburger" is 3 tokens. "hello" is 1. Punctuation, spaces, and special characters all count.

Why it matters

Every AI model has a context window — a maximum number of tokens it can process at once. Exceed it and your input gets cut off or rejected.

Costs money too

API pricing is per token. A document that looks small can contain thousands of tokens. Knowing this upfront helps you plan and avoid surprise bills.

Model differences

Each model uses a slightly different tokeniser. GPT and Claude are similar. Gemini uses a different method and typically produces fewer tokens for the same text.
